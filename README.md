# 6r - Simple Windows TUI File Editor

A lightweight C-based text editor for Windows command line with terminal user interface.

## Features

- Line-based text editing with line numbers
- Keyboard controls (arrow keys, Home, End, Page Up/Down)
- File operations (open, save, save-as, new)
- Status bar showing file info and cursor position
- Modular architecture
- Minimal dependencies, fast startup

## Requirements

- Windows OS (uses Windows Console API)
- C compiler
- Batch script support

## Build

```bash
build.bat run      # Test the editor
build.bat clean    # Clean build files
build.bat rebuild  # Recompile everything
```
